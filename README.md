# react-native-capped-scrollview

React Native ScrollView with a maxVelocity prop

## Installation


```sh
npm install react-native-capped-scrollview
```


## Usage


```js
import { CappedScrollviewView } from "react-native-capped-scrollview";

// ...

<CappedScrollviewView color="tomato" />
```
